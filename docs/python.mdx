## Lab 5: The Countdown

:::info Concept Definition: For Loop
A way to repeat a specific block of code a set number of times in a sequence.
:::

**The Mission:** Iteration is perfect for time-sensitive sequences like a 5-second countdown to ignition.


## Lab 8: Data Slicing

:::info Concept Definition: Slicing
A method used to extract a specific portion or "slice" of a list or sequence.
:::

**The Mission:** By using `[start:stop]`, we can isolate specific "windows" of mission data, like the last three sensor logs.

```python title="lab8_data_slicing.py"
sensor_logs = [10, 22, 35, 48, 55, 62, 70]
recent_data = sensor_logs[-3:]  # Extracts the last three entries
print(f"Recent trajectory markers: {recent_data}")
```

## Lab 9: Orbital Persistence

:::info Concept Definition: While Loop
A loop that repeats indefinitely as long as a specified condition remains `True`.
:::

**The Mission:** Used for continuous monitoring until a threshold — like stable orbit velocity — is reached.

```python title="lab9_orbital_persistence.py"
velocity = 25000
while velocity > 17500:
    print("Braking thrusters active...")
    velocity -= 2500
print("Orbit stabilized.")
```

## Lab 11: Fault Isolation

:::info Concept Definition: Exception Handling
A way to prevent a program from "crashing" when an error (exception) occurs.

## Lab 12: Automated Filtering

:::info Concept Definition: List Comprehension
A concise, shorthand way to create or filter lists in a single line of code.
:::

**The Mission:** This is an efficient way to process large amounts of sensor data, such as identifying only "hot" temperature zones.

```python title="lab12_automated_filtering.py"
temps = [22, 25, 18, 32, 21]
hot_zones = [t for t in temps if t > 25]
print(f"Alert: High heat in sectors: {hot_zones}")
```

## 🌟 Final Challenge: The Flight Director

Combine everything you've learned:

1. Store ship stats in a **Dictionary**
2. Create a **Function** that uses **Logical Operators** for a pre-flight check
3. Use a **For Loop** to countdown
4. Use an **If/Else** block to confirm ignition

```python title="final_challenge_flight_director.py"
# 1. Ship stats dictionary
ship = {"fuel": 80, "oxygen": True, "shields": 95}

# 2. Pre-flight check function
def pre_flight_check(stats):
    if stats["oxygen"] and stats["fuel"] > 50 and stats["shields"] > 80:
        return True
    return False

# 3. Countdown loop + 4. Ignition check
if pre_flight_check(ship):
    for second in range(5, 0, -1):
        print(f"{second}...")
    print("🚀 Ignition! Artemis One is go for launch!")
else:
    print("❌ Launch aborted. Review ship diagnostics.")
```
